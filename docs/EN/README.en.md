## Features 🚀

- _**Download**_ ↧: Download all the scans you desire automatically and in one click!
- _**Storage**_ 📚: Downloaded files are stored in the right place at the right time!
- _**Update**_ (Manual/Automatic) ♽: Real magic 🪄, stay up to date with the latest releases of your scans!
- _**Changelog**_ 🔗: Stay informed about changes in your data! All changes are recorded after an update 𝌤
- _**Configuration**_ ⚙️: Configure the essential aspects of the application as you like; PandaScan 🐼 is YOURS 🫵

## It's Simple 🧞‍♂️ 

_Make sure you're connected to the Internet!_

1. Launch the PandaScan application 🐼 (App.py)
2. Search for your favorite manga 🔍
3. Select the manga and chapters to download 🤩
4. Start the download: PandaScan takes care of the rest 💨
5. Read without moderation 🤓
<details>
        <summary> Optional ⚙️ (BETA only) </summary>

- Configure the **Download destination path** of your downloads (Default PandaScan directory)
- Choose to enable or disable the **Update** of the application (Enabled by default)
  - Choose between **Manual** or **Automatic** (Manual by default)
- Enable or disable the **Automated Browser View** (Disabled by default)
</details>
