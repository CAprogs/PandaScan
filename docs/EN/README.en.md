## Features 🚀

- _**Download**_ ↧: Automatically and with a single click, download all the scans you desire!
- _**Storage**_ 📚: Downloaded files are stored in the right place, at the right time!
- _**Update**_ (Manual / Automatic) ♽: A real magic trick 🪄, stay up to date with the latest releases of your scans!
- _**Changelog**_ 🔗: Stay informed about changes in your data! All changes are recorded after an update 𝌤
- _**Configuration**_ ⚙️: Configure the essentials of the application as you like, PandaScan 🐼 is YOURS 🫵

## It's Simple 🧞‍♂️ 

_Make sure you're connected to the Internet!_

1. Launch the PandaScan 🐼 application (App.py)
2. Search for your favorite Manga 🔍
3. Select the Manga and Chapters of YOUR choice 🤩
4. Start the download: PandaScan takes care of the rest 💨
5. Read without moderation 🤓
<details>
        <summary> Optional ⚙️ (Beta version only) </summary>
        
- Configure the **Destination path** for your downloads (Default is the PandaScan directory)
- Choose whether to enable the application **Update** or not (Enabled by default)
  - Choose between **Manual** or **Automatic** update (Manual by default)
- Enable or disable the **Automated Browser View** (Disabled by default)
      
</details>
